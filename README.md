VI - Manual by Fabio Szostak!
===================


VI command summary. Enjoy and increase it. You're welcome.

----------
**Open a file**

Press <kbd>Esc</kbd> and type
```
:e /path/of/filename
```
**Open a file ignoring the current changes**

Press <kbd>Esc</kbd> and type
```
:e! /path/of/filename
```
